# TaskMaster API

TaskMaster is a Node.js + Express REST API for collaborative task tracking.

## Quick Start

1. Install dependencies

```bash
npm install
```

2. Copy env template

```bash
cp .env.example .env
```

3. Migrate and seed database

```bash
npm run db:migrate
npm run db:seed
```

4. Run in development mode

```bash
npm run dev
```

API base path: `/api/v1`
